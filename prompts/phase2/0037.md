## 用語整理
### surrogate torque
actual motor torque ではなく、body の回転応答のみを切り分けるために与える**代理トルク入力**を指す概念名である。

### `body_equiv_load.pure_couple`
surrogate torque の具体実装の1つ。  
合力 0・非零トルクとなる 2 力を body に与える。Phase1 で用いる。

### torque split / force couple 分配
actual motor torque を、attach / first / second 近傍の bead 力へどう割り振るかの設計を指す。  
今回の主戦場の1つである。

### local constraint
hook / basal / first torsion 近傍に対して効く局所的な shape maintenance を指す。  
attach-first, first-second, basal bend, first torsion 近傍を主に扱う。

### canonical config
phase ごとに比較の基準として固定する標準設定。  
今回の比較は canonical config 上で行うこと。

### `minimal_basal_stub`
body + hook only ではなく、**3つの flagellar beads を含む短い basal stub** を持つ最小構成を指す。  
構造としては `attach(body) -> first -> second -> third` の最小 stub である。
