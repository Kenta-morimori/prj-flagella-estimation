## 目的
- 2-1で実装した物理コアを、Watari & Larson 2010 の条件（Fig.1, Table 1, Fig.3の例）に揃える。
- 出力仕様を確定：
  - 3D：追従カメラOK、連結ビーズ表示、べん毛ラベル、タイムスタンプ、全ステップをフレーム保存→mp4化
  - 2D：追従カメラNG（固定カメラ）、25Hzで間引き出力

## 出力仕様（追加要件）

### 3D（全ステップ）
- 3D描画は「実ビーズ連結」を必須：
  - body: prism beads を層内エッジ＋層間エッジで連結して表示（簡易なら“各層の多角形＋縦エッジ”）
  - flagella: べん毛ごとのビーズ列を線で連結
- べん毛は `F0, F1, F2` として凡例表示（ラベリング必須）
- 追従カメラ：中心=菌体中心（全body beads平均など）
- 表示範囲：`render.view_range_um`（±R）
- タイムスタンプ：左上 `t = {t:.3f} s`
- 全ステップ保存：
  - `dt` ごとに `frames_3d/frame_XXXXXX.png`
  - 最後に `movie_3d.mp4` を生成（PNG連番→mp4）

### 2D（固定カメラ・25Hz）
- 固定カメラ（追従禁止）：
  - 画面中心は常に固定、pixel_size_um固定
  - 画面外はクリップ（見切れを許容）
- 25Hz間引き：
  - `fps_out_2d = 25`
  - 2Dフレーム保存またはmp4生成（既存パイプラインに合わせる）

## conf/sim_swim.yaml（論文条件へ揃えた“確定値”）
重要：各キーにコメントで「意味・単位・論文対応」を書くこと。

以下の値で `conf/sim_swim.yaml` を更新する（no-wall v1）：

scale:
  b_um: 1.0                   # [µm] 長さスケール b（菌体長軸の半分）
  bead_radius_a_over_b: 0.1    # [-] a=0.1b（RPYに必要）

body:
  # 形状（Fig.1相当：三角柱=3角柱、層数5、層間隔0.5b、全長2b）
  prism:
    n_prism: 3                # [-] 断面の角数（デフォルト3）
    n_layers: 5               # [-] 軸方向の層数（3*5=15 beads）
    dz_over_b: 0.5            # [-] 層間隔（0.5b）
    radius_over_b: 0.5        # [-] 断面半径（中心→断面ビーズ距離。モデル定義を固定）
    axis: "x"                 # 軸方向
  length_total_um: 2.0        # [µm] 出力・表示用（(n_layers-1)*dz*b = 2b）

flagella:
  n_flagella: 3               # 論文条件
  placement_mode: "uniform"   # 終端層の断面ビーズから均等に付着点を選ぶ
  discretization:
    ds_over_b: 0.58           # べん毛の隣接間隔（Table 1のL=0.58bに合わせる）
  bond_L_over_b: 0.58         # spring平衡長（べん毛）
  # べん毛の初期形状（ヘリックス点列）は補助。多形変換は角度スイッチで再現。
  helix_init:
    radius_over_b: 0.2        # 必要なら調整（初期形状を安定にするため）
    pitch_over_b: 1.0

fluid:
  viscosity_Pa_s: 0.001        # 水相当（スケール決定に使う）

motor:
  torque_Nm: 4.0e-18           # [N·m] スケール。tau = eta*b^3/T で時間が決まる
  reverse_n_flagella: 1        # tumbleで反転するべん毛本数（最低1本）

potentials:
  spring:
    H_over_T_over_b: 10.0      # H = 10 T / b
    s: 0.1                     # 伸長限界（無次元）
  bend:
    kb_over_T: 20.0            # kb = 20 T
    theta0_deg:
      normal: 142.0
      semicoiled: 90.0
      curly1: 105.0
  torsion:
    kt_over_T: 10.0            # kt = 10 T
    phi0_deg:
      normal: -60.0
      semicoiled: 65.0
      curly1: 120.0
  spring_spring_repulsion:
    A_ss_over_T: 1.0
    a_ss_over_b: 0.2
    cutoff_over_b: 0.2

hook:
  enabled: true
  threshold_deg: 90.0
  kb_over_T: 20.0              # v1では同程度で開始（必要なら独立比へ）

run_tumble:
  # Fig.3の例（τ単位）
  run_tau: 1200.0
  tumble_tau: 800.0
  semicoiled_tau: 400.0
  curly1_tau: 400.0

time:
  duration_s: 5.0              # 追加要件：物理時間 5s
  dt_over_tau: 0.001           # Table 1のΔt（τ単位）

output_sampling:
  # 3D：全ステップ（dtごと）
  out_all_steps_3d: true
  # 2D：25Hz間引き
  fps_out_2d: 25.0

brownian:
  enabled: false               # デフォルトOFF（ON時はFDT整合）
  temperature_K: 298.0
  method: "cholesky"
  jitter: 1.0e-20

render:
  # 3D
  save_frames_3d: true
  follow_camera_3d: true
  view_range_um: 10.0
  timestamp_3d: true
  timestamp_fmt: "t = {t:.3f} s"
  label_flagella: true

  # 2D
  follow_camera_2d: false
  save_frames_2d: true
