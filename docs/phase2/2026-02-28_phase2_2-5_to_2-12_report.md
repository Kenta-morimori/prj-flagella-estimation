## 決定事項（仕様）
- ログは `outputs/<run_id>/sim/step_summary.csv` に統一し、`step_summary_full.csv` は廃止。
- MVP制約は `body.n_prism = 3`、`0 <= flagella.n_flagella <= 3`。
- `motor.torque_Nm = -1` は内部規約トルク、`0` はモータOFF、その他は物理トルクとして扱う。
- 内部時間積分は常に `tau_s = 1`、`dt_internal_s = 1e-3`、`dt_star = 1e-3`。
- `time.dt_s` は内部刻みではなく、設定上の出力間隔として保持する。
- body–flag 接続は spring + hook の二重拘束とする。
- フック長は 0.25b、flag intra の自然長は 0.58b。
- べん毛の論文固定値は radius 0.25b、pitch 2.5b、normal bending 142°、normal torsion -60°。
- Phase2 のデフォルトは run固定とし、`motor.enable_switching = false` を既定とする。

## 定義
- run固定: `flagella` の polymorph state が変化しないこと。判定は `flag_state_changed == False`。
- body長軸 `body_axis`: bodyビーズ座標の PCA 第1主成分（単位ベクトル）。
- 後方方向 `rear_dir`: `-body_axis`。
- 基部接線 `tangent0`: `normalize(flag1 - flag0)`。
- 後方整列条件: `angle(tangent0, rear_dir) <= 10°`。

## 実施内容（変更点サマリ）
- `sim_debug` 系の診断CSVを廃止し、`step_summary.csv` の単一時系列ログへ統合した。
- モータトルクの力分配を Fig.2 準拠の幾何学ベースへ更新し、モータ診断値（退化軸・rank不足・bond clipping）をログ化した。
- `render2d` のべん毛描画をデフォルトOFFにし、3D動画出力を `swim3d.mp4` に統一した。
- body–flag 接続を spring + hook とし、attach層を body中央層に変更した。
- spring自然長を全ペアで初期距離から再計算する方式に統一した。
- body に層内三角 bending と側面ブレースを追加し、さらに body剛体投影を導入して shape drift を抑制した。
- `dt_star=1e-3` 固定運用へ切り替え、内部刻みと出力間隔を分離した。
- hook長と flag intra 長を long-step で維持するため、距離投影と flagella テンプレート投影を導入した。
- べん毛初期点列を「radius 0.25b / pitch 2.5b / 隣接距離 0.58b」を満たす解析形 + Δx 二分法で生成するように変更した。

## 重要なアルゴリズム変更（要点）
- モータ力分配: フック近傍2ボンドへのトルク分解から力カップルを構成する方式へ変更。
- body剛体化: 強い body spring/bending に加え、Kabsch による body rigid projection を各step後に適用。
- hook/flag長維持: hook距離投影 + flagella テンプレート投影で、長ステップでも 0.25b / 0.58b を維持。
- helix生成: `d(Δx)=sqrt(Δx^2+(2R sin(pi Δx/p))^2)=L` を二分法で解き、`L=0.58b` を満たす点列を生成。
- 角度拘束: flagellum の平衡角は `θ0=142°`, `φ0=-60°` を直接適用し、state切替時のみ表の値へ切替。
- 後方整列: body PCA 軸から `rear_dir` を定義し、ローカル螺旋を `rear_dir` 基底へ回転して基部接線を整列。
- run固定: `motor.enable_switching=false` のとき `DynamicsEngine._update_run_tumble_state()` を実行しない。

### multi-step で assert していること
- body rigidity（1000 step, `motor=0`, `hook=false`）
  - `max(|L-L0|/L0) < 1e-4`
  - `max(|θ-θ0|) < 1e-3 rad`
- body stability（2000 step, `motor=1e-19`）
  - 全座標 finite
  - `max_stretch_ratio < 5e-4`
  - `max_angle_error_rad < 2e-3`
- hook length（2000 step, `motor=0` / `1e-18`）
  - `hook_count == n_flagella`
  - mean: `0.25b ±5%`
  - min/max: `0.25b ±15%`
- flag intra length（2000 step, `motor=0` / `1e-18`）
  - `flag_intra_count > 0`
  - mean: `0.58b ±2%`
  - min/max: `0.58b ±5%`
- flag angle error（2000 step, `motor=1e-18`）
  - `flag_bend_err_mean_deg <= 6`
  - `flag_bend_err_max_deg <= 20`
  - `flag_torsion_err_mean_deg <= 6`
  - `flag_torsion_err_max_deg <= 25`
- run固定（1000 step以上, `motor.enable_switching=false`）
  - `flag_state_changed` が全行 `False`
